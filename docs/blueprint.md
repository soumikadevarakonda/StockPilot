# **App Name**: StockPilot

## Core Features:

- Authentication UI: Login/Signup Page: Minimal form with card layout and background illustration.
- Dashboard UI: Dashboard Page: Top navbar with logo and user profile, portfolio summary cards (balance, invested amount, P/L).
- Market Data UI: Stocks Market Page: Searchable table of stocks (Name, Price, Change %, Buy/Sell buttons).
- Portfolio Holdings UI: Portfolio Page: Responsive table with holdings (stock, qty, avg price, current price, P/L).
- Trade Execution Modal: Trade Modal: Smooth animation modal for buying/selling stocks with quantity input and confirmation.
- Mock API Integration: Data fetching from mock data.
- Theme Switcher: Dark/Light Mode Toggle.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF) for a clean, trustworthy fintech look.
- Background color: Light gray (#F0F2F5) to maintain a bright and spacious feel in light mode, and dark gray (#333333) for dark mode.
- Accent color: Sea green (#2E8B57) for positive changes and calls to action; Salmon (#FA8072) for negative changes.
- Body and headline font: 'Inter' sans-serif for a modern, machined, objective, neutral look that suits both headlines and body text.
- Fintech dashboard layout with minimal colors (white, gray, blue, green, red for stock trends), rounded corners, and subtle shadows for depth.
- Use simple, outlined icons for navigation and interactive elements to ensure clarity and a modern aesthetic.
- Subtle Framer Motion animations for transitions and modals to enhance user experience and provide a smooth, engaging feel.
